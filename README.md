# Project Portfolio

A fullstack application with a Next.js frontend and a Node.js/Express/MongoDB backend.

## Structure

- `frontend/`: Next.js (React) application
- `backend/`: Node.js (Express) application using MVC structure

## Getting Started

### Backend Setup
1. `cd backend`
2. `npm install`
3. `cp .env.example .env`
4. `npm run dev`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## MVC Structure in Backend
- `models/`: Mongoose schemas
- `controllers/`: Request handlers
- `routes/`: API endpoint definitions
- `config/`: Configuration files (e.g., DB connection)
- `middleware/`: Custom Express middleware
